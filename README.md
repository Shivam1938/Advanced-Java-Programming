# Advanced Java Programming

This repository contains programs related to **Advanced Java Programming** for **B.Tech CSE (Semester 6)**.

## 📂 Topics Covered

### Unit 1

* Implementing Server
* Making URL Connection
* One Way Client-Server Communication
* Two Way Blocking Communication
* Two Way Multithreaded Communication

Each folder contains Java source code inside the **src** directory.
